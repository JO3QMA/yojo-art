# yojo-art
yojo-art/cherrypickのv2系と連携して動作するリバースプロキシ
https://github.com/yojo-art/cherrypick

# 使い方

## devcontainerの設定

1. `.config_example`フォルダーを`.config`としてコピーします：
```bash
cp -r .config_example .config
```

2. VSCodeでdevcontainerを開きます：
   - VSCodeでこのフォルダーを開く
   - 左下の「><」アイコンをクリック
   - 「Reopen in Container」を選択

これでdevcontainer環境が立ち上がり、開発を始めることができます。
