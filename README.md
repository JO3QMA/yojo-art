# yojo-art

## 概要

yojo-art/cherrypick v2と連携して動作するリバースプロキシです。
詳細は[cherrypickのリポジトリ](https://github.com/yojo-art/cherrypick)をご確認ください。

## セットアップ手順

### 1. 設定ファイルの準備

開発を始める前に、設定ファイルのコピーが必要です：

```bash
cp -r .config_example .config
```

※ この手順は必須です。設定ファイルがないと正常に動作しません。

### 2. devcontainer環境の起動

1. VSCodeでプロジェクトフォルダを開く
2. 左下の`><`アイコンをクリック
3. `Reopen in Container`を選択

## 機能

- cherrypick v2との連携機能
- リバースプロキシ機能
- その他機能は開発中です

## 開発について

devcontainer環境が起動したら、開発を開始できます。
不明点や問題がありましたら、Issueを作成してください。
